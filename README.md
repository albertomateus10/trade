# Plate Search Pro (Versão Python)

Esta é a versão de alta performance do buscador de placas, capaz de processar centenas de milhares de linhas quase instantaneamente.

## Como Instalar

1. Abra o Terminal (PowerShell ou CMD) na pasta do projeto:
   `cd "C:\Users\alber\.gemini\antigravity\scratch\plate_search_pro_python"`
2. Instale as bibliotecas necessárias:
   `pip install -r requirements.txt`

## Como Rodar Online (Link)

Este projeto está preparado para o **Streamlit Cloud**.
1. Suba o código para o seu GitHub.
2. Acesse [share.streamlit.io](https://share.streamlit.io).
3. Conecte seu repositório.
4. O link será gerado automaticamente (ex: `seu-app.streamlit.app`).

## Como Rodar Localmente (Web)

1. Instale o Streamlit:
   `pip install streamlit`
2. Execute o comando:
   `streamlit run streamlit_app.py`

## Como Rodar Localmente (Desktop)

Basta executar o comando:
`python main.py`
